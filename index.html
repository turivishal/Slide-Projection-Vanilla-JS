Slide Projection (SP) GitHub

Demos:

  • Next / Prev Controls - Set inside / outside the container, Create multiple controls for single slider.
  • Pause on Hover Controls - Slider pause on mouse enter and play on leave.
  • Caption Controls - Slide caption at bottom and customization.
  • Manual Fx Controls - Slide in / out effect customization, adding custom class.
  • Overlay Options - Title and description of slide customization.
  • Speed Option - Set interval time in milliseconds.
  • Auto Play Option - Set auto play on / off slider.
  • Multiple Slideshow in Single Page - above almost demo examples have doubled sliders.
